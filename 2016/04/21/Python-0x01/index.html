一些简单的入门知识点

  • 跟matlab类似直接输入代码，回车得到执行结果
  • python程序以.py结尾windows下不能双击运行
  • 输入和输出

    input()可以用来接收输入

    在交互命令行中也可以直接>>>name输出name的值 和matlab一样
    提示信息可以在input()里输出

  • 其他

    #用来注释，以:结尾的语句之后连续缩进的语句为一个代码块
    貌似用四个空格缩进最稳妥o-o

数据类型和变量

  • 整数
    Python可以处理任意大小的十进制整数，也可以使用十六进制
  • 浮点数
    浮点数可以用数学写法如1.23，-9.56等等，对于需要表示成科学记数法的数可以用e代替x10,如1.23e9，12.3e-4等等
  • 字符串
    字符串是以"或'括起来的任意文本，在文本中若是需要出现'或"时可以用转义字符\标识

和其他语言一样转义字符\有很多作用，\n表示换行，\t表示制表符，但是Python中可以用r''表示''内部的字符不转义

  • 布尔类型

    布尔类型只有True、False两种值（要注意大小写），也可以用布尔运算运算出来，还可以用and，or，not运算 

    >>> True
True
>>> 3 < 2
False
>>> not False
True
  • 空值

    空值是Python里一个特殊的值，用None表示

  • 变量
    变量名必须是大小写英文、数字和_的组合，且以英文字母或_开头
    =表示赋值语句，可以把不同的数据类型赋值给变量，同一个变量可以被反复赋值 
    a = 123 #a是整数
print(a)
a = 'ABC' #a变为字符串
