  • const修饰符

    const主要被用于修饰函数的参数，使用const，意味着使其只读。

    上面的例子表明的只是src指针所指向的对象被定义为只读，而不是src这个变量。 

    char *strpy(char *dest, const char *src)
{
 src = NULL;  //<-对src赋值，编译器没有放错
 *src = NULL;  //<-编译器报错
}
